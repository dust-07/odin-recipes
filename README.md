# odin-recipes

## Description
This is a basic recipe website built as part of **The Odin Project Foundations** course.


## Goal
The goal of this project is to practice:
- HTML fundamentals
- Git and GitHub 

## Skills Practiced
- Organizing files and folders for website development
- Adding headings, paragraphs, links, images, and lists in HTML
- Basic git commands (`add`,`commit`,`push`)
